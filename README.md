# 🌿 Blooming Oasis - Premium Indoor Plant Store

Blooming Oasis is a polished, professional 4-page web application designed for a premium indoor plant store. Built with a focus on elegant design and user interactivity, it provides a seamless experience for plant enthusiasts to discover, learn about, and shop for their perfect green companions.

## 🚀 Live Preview
You can view the live application here: [Blooming Oasis Live](https://ais-pre-s2yixnxwrnuagcqo5etehq-753964473130.asia-southeast1.run.app)

## ✨ Key Features

### 1. Home Page
- **Plant Matchmaker Quiz**: An interactive tool that recommends the perfect plant based on your sunlight, care level, and pet-friendly needs.
- **Bento-style Photo Collage**: A visually stunning gallery showcasing our vibrant plant collection.
- **Curated Collections**: Easy navigation for beginners, air-purifying enthusiasts, and pet owners.
- **Dynamic Date Display**: Shows the current date, welcoming visitors with a personalized touch.

### 2. Product Detail Page (Monstera Deliciosa)
- **Deep Dive Info**: Comprehensive details on light, water, humidity, and pet safety.
- **Video Integration**: A dedicated care guide video to help users succeed with their new plant.
- **Customization Options**: Select potting types and add-ons like walnut wood stands.
- **Community Wisdom**: A tips section where users share their plant parenting experiences.

### 3. About Us Page
- **Narrative Storytelling**: Learn about the founders' journey from a sun-drenched apartment to a national mission.
- **Sustainability Focus**: Details on our eco-friendly sourcing and plastic-free shipping practices.
- **Meet the Team**: Faces behind the brand, including our lead botanist and plant stylists.

### 4. Contact Page
- **Live Chat Simulation**: Real-time interaction with a "Plant Specialist" for immediate care advice.
- **Modern Contact Form**: A professional form for general inquiries with a custom success modal.
- **Social Links**: Direct access to our Instagram, Pinterest, and TikTok communities.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure for all pages.
- **CSS3 (Bootstrap 5.3.3)**: Utilizes the latest Bootstrap framework for responsive, grid-based layouts and pre-styled components.
- **Custom CSS**: Premium typography (Playfair Display & Lato), custom animations, and a refined "creamy" aesthetic.
- **Vanilla JavaScript**: All interactivity (Quiz, Chat, Form handling, Scroll effects) is built with clean, modern JS.
- **Vite**: Modern frontend build tool for lightning-fast development and optimized production builds.

## 📂 Project Structure

```text

├── index.html          # Homepage

├── about.html          # About Us page
├── contact.html        # Contact page
├── src/
│   ├── main.js         # Core logic (Quiz, Chat, Date, Scroll)
│   └── style.css       # Custom premium styling
├── metadata.json       # Applet metadata
└── package.json        # Project dependencies
```

## 📖 How to Run
1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Build for production: `npm run build`
